Magic Whisk

Batch image generation for Google Whisk — queue prompts, auto-download results, browse in a built-in gallery.

View on GitHub

Free & open source. Install from source or Chrome Web Store.

What it does

📝

Batch Queue

Queue multiple prompts at once. Magic Whisk runs them one by one through Google Whisk automatically.

⬇️

Auto-Download

Every generated image is saved to your downloads folder instantly. No manual right-click saving.

🖼️

Built-in Gallery

Browse all generated images in a grid. Select multiple and batch download with one click.

⏸️

Pause & Resume

Pause the queue anytime. Resume right where you left off. Stop completely if you change your mind.

⚙️

Configurable

Set delay between generations, choose download folder, toggle auto-download on or off.

🛡️

Privacy-First

Runs entirely in your browser. No external servers, no data collection. Uses your existing Google login.

How it works

1

Open Google Whisk

Navigate to labs.google/fx/tools/whisk. Magic Whisk appears as a small icon on the right edge.

2

Add your prompts

Click the icon to open the panel. Type one prompt per row. Add as many as you want.

3

Hit Start

Magic Whisk takes over — injecting each prompt into Whisk, clicking generate, and collecting every result image.

4

Sit back

Images auto-download as they complete. Browse everything in the gallery tab when you're done.
